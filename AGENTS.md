# For agents and new contributors

- Consult **`docs/`** first for project knowledge and API notes.
- **Repo structure and conventions:** [docs/repo-conventions.md](docs/repo-conventions.md).
- **`tmp/`** is a sandbox; contents are not official mods.
- **`tools/`** scripts are part of the standard workflow (link, zip, tail-log, find_unused_mods).

## Trello-Hubs (board "My Life") — check BEFORE starting or carding work

This repo's work is tracked on Trello. Before building a feature or creating a new
card, check these hubs (and their sub-cards) so we don't duplicate planned work:

- **[HoldToSteer: ModHub Release](https://trello.com/c/szcRu53Q)** (#74) — the mod's hub; sub-cards cover bugs, features and release steps (label `FS25 Mod`).
- **[FS25 Mod-Set-Manager / fsmods](https://trello.com/c/AQiwTTVo)** (#139) — the per-savegame mod-set tool in `tools/fsmods/`.
