# Screenshots für ModHub

Dieser Ordner enthält In-Game-Screenshots für den ModHub-Eintrag.

## Benötigte Screenshots:

1. **screenshot1.png** - Route-Linie auf der Straße (grüne Linie sichtbar)
2. **screenshot2.png** - HUD-Anzeige mit Zielname und Distanz
3. **screenshot3.png** - Abbiegungsanzeige ("in Xm links/rechts")
4. **screenshot4.png** - Vogelperspektive mit vollständiger Route
5. **screenshot5.png** - Verschiedene Situationen (Brücke, Feld, etc.)

## Format:
- Auflösung: 1920x1080px (Full HD)
- Format: PNG oder JPG
- Dateigröße: < 5MB pro Bild empfohlen

## Aufnahme-Tipps:
- Verwende die beste Grafikqualität im Spiel
- Zeige die Navigation in verschiedenen Situationen
- Stelle sicher, dass die Route-Linie gut sichtbar ist
- Zeige die HUD-Anzeige klar und lesbar
